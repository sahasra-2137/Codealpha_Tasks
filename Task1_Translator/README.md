## How It Works
1. User enters text or uses voice input
2. The system automatically detects the source language (optional).
3. Romanized Indian language text is converted into native script using transliteration.
4. The text is sent for translation.
5. The translated output is displayed instantly.
6. Users can listen to the translated text, copy it, or review it later from translation history.
